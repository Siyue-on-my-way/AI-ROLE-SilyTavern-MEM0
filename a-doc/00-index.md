# role_chat 文档索引

更新时间：2025-12-28

本目录用于记录 role_chat 项目（SillyTavern + mem0）集成的需求、业务、架构与待办，目标是在保持开源项目整洁（便于跟随上游升级）的前提下，实现“长期记忆（mem0）”能力的可插拔接入。

## 文档列表

- `01-PRD.md`：需求 PRD（目标、范围、验收标准、非功能需求、约束）
- `02-业务设计.md`：业务流程与交互（写入记忆、检索增强、注入提示词、失败回退）
- `03-架构设计.md`：微服务与部署架构、接口契约、配置、数据存储、运行拓扑
- `04-待开发事项.md`：待开发事项/里程碑/风险与决策记录（ADR 形式的要点）

## 核心原则（面向上游升级）

- 尽量不修改上游源码：优先“配置化/外置服务/可拔插扩展/patch 最小化”。
- 前端界面尽量不变：确需新增设置项时，集中在现有设置页，避免大改 UI 框架与交互。
- 部署微服务化：SillyTavern 与 mem0 各自独立容器运行，经 HTTP 通信；数据库组件共用一套。
